Hunted Tiles⌗

GitLab Repository ⧉ | GitHub Mirror Repository ⧉

Hunted Tiles is my custom Linux window manager environment using Lua, AwesomeWM framework, and shell scripts to integrate the dynamic tiling window manager with other essential tools. From system monitors like battery, volume, brightness, CPU & memory widgets to integration with Vim and other terminal based applications, Hunted Tiles try to be a complete desktop environment without requiring many dependencies and allow users to choose dependencies. The name “Hunted Tiles” was inspired by my favorite color scheme (and main theme of Hunted Tiles), Dracula, and the fact that AwesomeWM is a tiling WM.

i3 & Sway Tips⌗

GitLab Repository ⧉ | GitHub Mirror Repository ⧉

Collection of commands, keybindings, and recommended applications for i3 or Sway window manager beginners. Some of the recommendations are subjective and does not contain all the information you might need, but I would love to make this a community project so feel free to suggest or correct information through issues or to me personally! If you love Microsoft, you can read this in the GitHub mirror repository instead of in GitLab.
